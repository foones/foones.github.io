ŅQué son?

Un conlang, es un lenguaje construído por humanos artificialmente (del inglés constructed language), y que puede ser desde un simple hobby hasta una compleja tarea.

ŅPor qué y para qué?

El por qué de hacerlo, tiene que estar estrictamente decidido por vos. Si sos creativo, tendrías que -al menos- comprenderlo. Si no sos creativo por naturaleza, de todos modos tenés muchas razones para justificarlo. Es decir que hay muchas razones para intentar crear un conlang. Puede que te guste escribir y, creando culturas para tus cuentos no quieras que carezcan de lenguaje. Es posible que lo hagas para jugar un juego de rol, para esconder o envíar mensajes a quien quieras, o simplemente por diversión.
En cuanto al objetivo, es decir a la utilidad, depende de para que se use, pero si no estás dispuesto a crear por el crear mismo, no creo que lo tuyo sea hacer conlangs ("conlangear").
Si no querés crear un lenguaje muy complejo, podrías desarrollar solamente un diccionario de nombres, para tus personajes, o para nombres geográficos. Para animarte a empezar, podés crear palabras nuevas para tu idioma nativo y utilizarlas para complementarlo; podés cambiar levemente tu idioma nativo en cuanto a fonética o gramática; podés hacer una mezcla entre dos idiomas naturales o muchas otras opciones.

ŅCómo?

Podés creer que para crear un lenguaje es necesario mucho conocimiento. Esto no es del todo falso, pero no implica que debas ser alguien especial para crear un lenguaje.
Cualquiera puede hacerlo.

Hay pasos a seguir para crear un lenguaje. Obviamente podés seguir los pasos que quieras pero tenés que tener en cuenta que un idioma no se compone sólo de palabras sino también de gramática, es decir sintaxis y morfología estables; y tienden a tener una estructura de palabras homogénea.
Los pasos que se suelen recomendar son:

  1. Establecer los fonemas y crear una estructura silábica
  2. Crear las palabras [a veces postpuesto]
  3. Crear una sintaxis y morfología
  4. Crear un alfabeto [optativo]
  5. Hacer evolucionar el lenguaje para
    que parezca natural

No esperes terminar tu lenguaje!!!
Un lenguaje real (es decir natural), permanece en constante evolución -mientras no esté muerto- y así debe estarlo el tuyo.
Para completar un lenguaje, es decir, dar la evolución del último paso, sería interesante que tuvieras al menos dos lenguajes: uno vivo y el otro muerto del que se extraigan raíces o ambos vivos y en constante unificación.
